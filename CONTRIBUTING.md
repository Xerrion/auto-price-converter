# Contributing

Thanks for contributing! This project welcomes issues and PRs.

## Ways to help

- Report bugs and suggest features
- Improve docs or examples
- Add tests or fix regressions

## Development setup

1. Install dependencies
2. Run the dev build

## Commands

- `bun run dev` — start Vite dev server
- `bun run build` — build for production
- `bun run test` — run tests (watch)
- `bun run test:run` — run tests once
- `bun run check` — typecheck and Svelte checks

## Pull request guidelines

- Keep PRs focused and small when possible
- Add or update tests where it makes sense
- Update documentation when behavior changes
- Ensure all checks pass before requesting review
